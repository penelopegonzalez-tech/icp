¿Cómo funciona el Factor Delta Δ?
Medición base
Primero capturas y calculas el ICP normal. Ese es tu punto de partida. Ejemplo: ICP 38.5
Cortes de actualización
Ingresas el total acumulado de cada variable en ese corte. Si antes tenías 45% neg y subió a 63%, escribes 63. El sistema calcula el incremento automáticamente.
ICP recalculado
El sistema calcula el ICP con los valores totales y muestra cuánto subió o bajó vs el corte anterior (Δ).
Alerta automática
Si el ICP sube ≥20 pts en un corte, se activa 🚨 Alerta Delta — ignición rápida, acción inmediata.
⚡ Ejemplo de uso (Opción A — valor total acumulado)
Base 10:00 → Neg 20% · Repros 500K → ICP 25.1 (GÉLIDO)
Corte 2 12:30 → Neg 38% (+18pp) · Repros 1.2M → ICP 38.5 Δ+13.4
Corte 3 14:00 → Neg 55% (+17pp) · Repros 2.8M → ICP 58.2 Δ+19.7
Corte 4 16:00 → Neg 72% (+17pp) · Repros 4.1M → ICP 79.0 Δ+20.8 🚨
